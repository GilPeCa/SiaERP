# TechnicalTest
ERP Test
API para gestion de inventario desarrollada para una prueba tecnica. Administra productos y revisa stock utilizando local db.

## Instalacion y ejecucion
1. Clona repo
2. Crear virtual environment "python -m venv venv" y activarlo 
3. Activar entorno virtual e instalar las dependencias
pip install -r requirements.txt
